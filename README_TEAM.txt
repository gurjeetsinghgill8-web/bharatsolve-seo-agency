  📥 HOW TO INSTALL (First Time Only)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Install Python
━━━━━━━━━━━━━━━━━━━
  → Go to: https://python.org/downloads
  → Download Python 3.12 or 3.11
  → INSTALL it (IMPORTANT: check "Add Python to PATH" during install)
  → Restart your laptop after installing

Step 2: Extract this ZIP
━━━━━━━━━━━━━━━━━━━━━━━━
  → Right-click the ZIP file → "Extract All..."
  → Choose any folder (e.g., Desktop or Documents)
  → You'll see these files inside:

      BHARATSOLVE_Launcher.bat     ← Use this to START the app
      BHARATSOLVE_Setup_Keys.bat   ← Use this to ADD your API keys
      app.py                       ← Main app (don't touch)
      secure_vault.py              ← Security system (don't touch)
      ... (other files)

Step 3: Get Free API Keys
━━━━━━━━━━━━━━━━━━━━━━━━━━
  You need FREE API keys for the AI to work:

  [1] GROQ API KEY (FASTEST - Recommended)
      → Go to: https://console.groq.com
      → Sign up with your Google/GitHub account
      → Go to API Keys section
      → Click "Create API Key"
      → Copy the key (starts with "gsk_...")

  [2] GEMINI API KEY (FREE - Google)
      → Go to: https://aistudio.google.com/apikey
      → Click "Get API Key"
      → Copy the key (starts with "AIza...")

  💡 Just Groq key is enough to start! Gemini adds more features.

  1️⃣  Double-click:  BHARATSOLVE_Setup_Keys.bat
      → Paste your API keys when asked
      → Your keys get ENCRYPTED and locked to your laptop
      → No one can steal them!

  2️⃣  Double-click:  BHARATSOLVE_Launcher.bat
      → Press 1 and Enter
      → Your browser will open automatically

  4️⃣  Start adding clients and using AI agents!

  ⏹️  To STOP the app: Close the black window, or press Ctrl+C

  → Just delete the whole folder, extract ZIP again, start fresh
